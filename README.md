**Wiki Journey** tracks your daily Wikipedia adventures and archives them in a tree format.

Get it from [Chrome Web Store](https://chromewebstore.google.com/detail/wiki-journey/lehenbcbjcnkhkikgopniimobmmdcfog) or [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/wiki-journey/)!
